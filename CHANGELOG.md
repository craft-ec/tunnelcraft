# Changelog

All notable changes to TunnelCraft will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Core Rust daemon with P2P networking
- Reed-Solomon erasure coding (5/3)
- Chain signature system for work verification
- Destination verification for trustless routing
- libp2p integration with Kademlia DHT
- Electron desktop app (macOS, Windows, Linux)
- iOS app skeleton with Network Extension
- Solana settlement contracts
- UniFFI bindings for mobile

### In Progress
- Mobile app completion
- Android VpnService integration
- Mainnet settlement contracts

## [0.1.0] - Unreleased

### Added
- Initial release
- Basic tunneling functionality
- Desktop app alpha
